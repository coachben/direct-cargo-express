  • Padding: Top/Bottom 80px
  • CSS Class: services-section

5 Add Header Content

Add three widgets in order:

  1. Heading: "WHAT WE OFFER" ‚Üí CSS Class: services-label
  2. Heading: "Explore Our Services" ‚Üí CSS Class: services-title
  3. Text Editor: Description ‚Üí CSS Class: services-description

6 Create 3-Column Grid

Add an Inner Section with 3 equal columns (33.33% each)

Add CSS Class: services-grid

7 Build Service Cards

For each of the 3 columns, add:

  • Container with class: service-card
  • Image (truck/plane/ship) with class: service-image
  • Icon overlay with class: service-icon-overlay
  • Content container with class: service-content
  • Title, description, and "Read More" link

8 Publish & Test!

Click Publish in Elementor and view your page!
